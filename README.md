Petal Pages is a full-stack personal journaling and lifestyle management web application built
using PHP, MySQL, HTML5, CSS3, and JavaScript. Designed with a soft, aesthetic visual
identity inspired by botanical themes and pastel colour palettes, the application provides
users with a comprehensive set of digital self-care tools in a unified, user-friendly interface.
The project aims to offer users a private, structured space to record daily thoughts, monitor
their emotional well-being over time, manage tasks and habits, and personalise their
experience through a sticker placement studio. Every feature is anchored to a date-based
dashboard that acts as the central hub of the application.
